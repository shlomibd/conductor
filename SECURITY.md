## Reporting a vulnerability

Please report security issues for Conductor using https://github.com/conductor-oss/conductor/security/advisories/new

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |
